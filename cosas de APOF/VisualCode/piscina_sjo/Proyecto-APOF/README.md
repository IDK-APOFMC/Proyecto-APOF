# 📰 Academia News

Proyecto académico de portal de noticias desarrollado con Vanilla JavaScript aplicando principios de arquitectura de software profesional.

## 🚀 Tecnologías

- HTML5 Semántico
- CSS3 (Grid, Flexbox, Variables)
- JavaScript ES6+ (Módulos, Async/Await, Clases)

## 🏗️ Arquitectura (MVC)

El proyecto implementa el patrón Modelo-Vista-Controlador adaptado al frontend:

- **Models**: Gestionan el estado y los datos de las noticias.
- **Views**: Únicas responsables de manipular el DOM y capturar eventos del usuario.
- **Controllers**: Orquestan el flujo de datos entre Modelos y Vistas.
- **Services**: Aíslan la lógica de llamadas a APIs externas.

## ⚙️ Instalación y Uso

1. Clonar el repositorio: `git clone https://github.com/IDK-APOFMC/Proyecto-APOF.git`
2. Abrir en Visual Studio Code.
3. Iniciar con la extensión **Live Server** (Requerido para soporte de ES Modules y ver la página).
